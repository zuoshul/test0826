In a word,
总之，
例句：In a word, we should study hard.
总之，我们要好好学习。

Apart from this，
除此之外
例句：Apart from this, I also like reading
除此之外，我还喜欢读书

From now on,
从今往后
例句：From now on, we need to study hard
从今往后，我们需要好好学习

On the one hand,...,on the other hand,...
一方面……，另一方面……
例句：On the one hand, we need to be helpful, on the other hand, we should be on the alert to protect ourselves.
一方面我们要乐于助人，另一方面我们要提高警惕保护自己。

in fact,
事实上
例句：In fact, I worked hard
事实上，我很努力

As we all know,
众所周知
例句：As we all know, China is a big country
众所周知，中国是个很大的国家

Thanks to / Because of...
因为.../ 感谢...
例句：Thanks to my mom, I become an excellent person.
因为我妈妈，我成为了一个优秀的人。

With the development of...
随着什么什么的发展。。。。
例句：With the development of the country
随着国家的发展

Anyway,
不管怎么样...
例句：Anyway, we are brothers.
不管怎么样，我们都是兄弟

There is no doubt that...
毫无疑问...
举例：There is no doubt that i like reading very much
毫无疑问，我非常喜欢读书

I suggest...
我建议我们应该
举例：I suggest we keep quiet
我建议我们应该保持安静

Finally,
最后，
举例：Finally, I want to say that we need to work hard
最后，我想说的是我们需要努力

The best way to solve this problem is...
解决这个问题的最好办法是
举例：The best way to solve this problem is to keep silent
解决这个问题的最好办法是保持沉默

As far as I'm concerned,
就我而言,
举例：As far as I'm concerned, it's time to go home. 
就我而言，我觉得该回家了。

Some people think....others think....
一些人认为....另一些人认为....
举例：Some people think we need to work hard, others think we don't need to work hard
一些人认为我们要努力，另一些人认为我们不需要努力
